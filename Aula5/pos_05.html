1. Controler

https://docs.flutter.dev/cookbook/forms/retrieve-input

1.1. TextEditingController

TextEditingController é um objeto que controla o texto dentro de um campo de texto TextField.

1.1.1. Criando TextEditingController

TextEditingController controlador = TextEditingController();

1.1.2. Ligar controlador ao campo

TextField(
	controller: controlador,
),

1.1.3. Ler o valor

print(controlador.text);

numero = int.parse(controlador.text);

1.1.4. Atualizar o valor do campo de texto

controlador.text = '$resultado';

controlador.text = '';

1.2. Passos

  1. Criar um controlador TextEditingController
  2. Ajustar o controlador de TextField
  3. Mostrar o valor atual do controlador

1.3. Exercício

  1. Crie um novo projeto, copie o código main.dart do GitHub, abra o menu View->Problems e faça todos os 3 TODOS, apague os comentários quando terminar.

2. Trabalhando com abas

As abas são um padrão de design comum em aplicativos móveis. Navegamos por diferentes partes da app usando a organização em abas. Para criar abas em Flutter usamos DefaultTabController para saber em qual aba estamos e TabBarView para indicar o conteúdo de cada aba.

2.1. Criando DefaultTabController

Usamos o construtor DefaultTabController (ver vídeo!) 

return MaterialApp(
  home: DefaultTabController(
    length: 2, // quantidade de abas
    child: Scaffold(

2.2. Mostrar abas

Criamos TabBar que contém as abas (tabs). Cada aba é criada com Tab. 

appBar: AppBar(
  bottom: const TabBar(
    tabs: [
      Tab(
	      text: 'Soma',
      ),

2.3. Conteúdo

O conteúdo é exibido conforme a ordem em que aparece em TabBarView 

body: const TabBarView(
  children: [
    Text('Soma'),
    Text('Subtração'),
  ],
),

2.4. Passos

  1. Usar DefaultTabController como home de MaterialApp.
  2. Criar as abas com TabBar dentro de appBar.
  3. Ajustar o conteúdo de cada aba com TabBarView no body.

2.5. Exercício

  1. No projeto https://github.com/maurojh/Flutter_Aula5_2024_pr2 crie a aba subtração e adicione a aba multiplicação.
